# 📊 전국 교통사고 데이터 분석 웹서비스

## 1. 데이터 분석 프로젝트 설명
본 프로젝트는 전국 교통사고 데이터를 기반으로  
**데이터 탐색, 시각화, 연도별 분석, 항목별 비교 분석을 할 수 있는 Streamlit 웹서비스**입니다.

사용자는 웹 화면에서 연도를 선택하거나 분석할 컬럼을 선택하여  
사고 건수·사망자수·부상자수 등 다양한 지표를 시각적으로 확인할 수 있습니다.

주요 기능:
- 데이터 미리보기  
- 연도 선택 + 컬럼 선택 분석  
- 연도별 전체 통계 분석  
- 그래프/테이블 기반 시각화  

---

## 2. 주제  
**전국 교통사고 데이터를 활용한 연도별·유형별 분석 및 시각화 웹서비스 구축**

---

## 3. 주제 선택 이유
- 교통사고는 사회적 중요도가 매우 높은 데이터  
- 연도/유형별 패턴 분석의 가치가 큼  
- 단순 분석이 아닌 ‘인터랙티브한 대시보드’ 구축 경험  
- Streamlit 기반 데이터 시각화 실습에 최적  
- 실제 정책·교육·연구 자료로도 활용 가능  

---

## 4. 데이터 분석 내용

### ✔ 데이터 출처
- 도로교통공단 TAAS 교통사고 통계  
- CSV 파일명: **10_24_stt.csv**

### ✔ 데이터 주요 컬럼
| 연도 | 대상사고 구분명 | 법정동코드 | 사고건수 | 사망자수 | 부상자수 | 치사율 | 구성비 등 |

---

### ✔ 웹앱 주요 페이지 구성

---

### **1) 🏠 HOME – 데이터 미리보기**
- CSV 로드 및 상위 데이터 표시  
- 데이터 구조 확인용 테이블 제공  
- Streamlit 성공 메시지 출력  

---

### **2) 📊 연도 + 컬럼 선택 분석**
기능:
- 연도 선택 → 해당 연도 데이터 필터링  
- 선택한 컬럼(사고건수/사망자수 등) 분석  
- 바 차트 시각화  
- 선택 데이터 테이블 제공  

출력 예시:
- “2010년 사고건수 그래프”
- 사고 구분명별 막대 그래프  
- 선택한 연도 데이터 목록  

---

### **3) 📈 연도별 전체 사고 분석**
제공 기능:
- 연도별 사고건수·사망자수·부상자수 통계표  
- 연도별 추세선(Line chart)  
- 2010~2019년 사고 지표 변화 시각화  

그래프:
- 사고건수(파란색)
- 사망자수(초록색)
- 부상자수(빨간색)
